`js/` -> all the javascript files.
`styles/` -> all the css
`html/` -> all the html

Their contents:

## JS/
- billboard.js -> script for the billboard
- main.js -> electron entry point.
- manager.js -> manager script.
- preload-manager/billboard/splash.js -> electron preload scripts.
- splash.js -> the splash's js script
- update-checker -> update search logic.
## HTML/
- billboard.html -> the billboard's html file
- manager.html -> the manager's html file.
- splash -> the splash screen's html file.
## STYLES/
- billboard.css -> css for the billboard
- manager.css -> css for the manager.
- splash.css -> css for the splash screen.
