`icons/` -> the app icons
`sounds/` -> the app sounds
`DSCPconfig.txt` -> config for the Device Specification Compatability Prompt (DSCP), it warns about your device's specs being too low to run the app(quite rare, the app is well-optimised). WN -> the prompt will apear when needed, ALLWAYS->it will apear allways, and NEVER...well...never.
`vrsn.txt` -> the app version, used in the splash.
